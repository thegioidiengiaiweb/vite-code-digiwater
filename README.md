# DigiWater Website

Website chính thức của **DigiWater – Viện nghiên cứu, kiểm định và phân tích nguồn nước**.

## Công nghệ sử dụng
- HTML5
- CSS3 (Tailwind CSS via CDN/CLI)
- Vanilla JavaScript

## Cấu trúc dự án
- `index.html`: Trang chủ
- `about.html`: Giới thiệu
- `services.html`: Dịch vụ xét nghiệm
- `contact.html`: Liên hệ
- `css/styles.css`: Stylesheet chính
- `js/main.js`: Logic xử lý giao diện

## Cách chạy local
Mở file `index.html` trực tiếp trên trình duyệt hoặc sử dụng Live Server.

## Deploy
Dự án đã sẵn sàng để deploy lên Vercel.
